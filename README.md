# Lista-de-Tarefas-React

 Projeto criado para relembrar a estruturação da arquitetura do REACT, utilizando os components para fazer a implementação de uma lista de tarefas bem simples.
 Onde a mesma apenas salva as tarefas utilizando o próprio LocalStorage, para armazená-las.

 Esse projeto é apenas uma prática de estudo, para me auxiliar a compreender e práticar REACT
